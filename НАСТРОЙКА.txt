ГРАФИК СМЕН — ЗАЛИТЬ НА GITHUB (один раз, ~5 минут)
====================================================

После заливки сайт будет тут:
https://vaflya67.github.io/grafik-smen/

Откроешь с телефона — календарь смен всегда под рукой.
Настройки хранятся в браузере на каждом устройстве отдельно.

ШАГ 1. Создать репозиторий
--------------------------
1. https://github.com → войди в аккаунт vaflya67
2. «+» → New repository
3. Имя: grafik-smen
4. Public → Create repository

ШАГ 2. Загрузить файлы
----------------------
1. На странице нового репо: «uploading an existing file»
2. Перетащи ВСЁ из папки grafik-smen на рабочем столе:
   - index.html
   - app.js
   - style.css
   - .gitignore
   - .nojekyll
   - этот файл НАСТРОЙКА.txt (не обязательно, но можно)
3. Commit changes

ШАГ 3. Включить GitHub Pages
----------------------------
1. Settings → Pages (слева)
2. Source: Deploy from a branch
3. Branch: main → Folder: / (root) → Save
4. Подожди 1–2 минуты

ШАГ 4. Проверить
----------------
Открой: https://vaflya67.github.io/grafik-smen/

На телефоне: Safari/Chrome → «Добавить на главный экран» — как приложение.

ОБНОВЛЕНИЕ
----------
Если что-то поменял в файлах — снова залей через GitHub
(Upload files или GitHub Desktop, если поставишь).
